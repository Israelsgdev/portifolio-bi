Dashboard desenvolvido para rede varejista com mais de 100 lojas, permitindo análise de performance por diversos ângulos.

  • Análise comparativa por período

  • Indicadores de desempenho (KPIs)

  • Integração com banco de dados SQL Server

Power BI DAX Power Query SQL
